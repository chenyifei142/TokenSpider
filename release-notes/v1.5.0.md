更新类型：功能新增与功能优化

### 新增

- 全面改用 Codex 风格单面板界面，并提供浅色、深色和跟随 Windows 系统三种主题模式；旧配置继续默认深色。
- 主面板与设置页均可即时切换主题，悬浮球、热力图、更新窗口、菜单和工具提示会同步刷新。

### 优化

- 主面板调整为固定信息层级：顶部核心指标与近 7 天费用、中部年度 Token 活动、底部使用统计；移除卡片拖拽排序。
- 近 7 天费用趋势改为七日柱状图，保留日期、金额坐标、悬停提示和空数据表达。
- 统一浅色与深色模式下的文字层级、焦点状态、边界、状态色和图表配色。
- 修复热力图提示背景透明、统计列未居中和面板底部圆角被内容层覆盖的问题，并缩小主题切换控件。
- 悬浮球默认尺寸由 96px 调整为 88px；旧版默认尺寸会自动迁移，自定义尺寸保持不变。

### 验证

- 补充主题配置、系统主题切换、即时持久化和已打开窗口原地换肤的回归测试。
- 补充固定布局、柱状图数据/悬停/零值、热力图双主题和设置主题控件测试。
- 在 820×550 与 640×550 面板、88px 悬浮球及设置/更新窗口上执行浅色和深色视觉对比。
