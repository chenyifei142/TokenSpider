更新类型：功能新增、Bug 修复与功能优化

### 新增

- 设置中新增「点击面板外部时自动收起」开关，默认开启；可按使用习惯关闭。
- Cookie 或 Token 失效时显示系统托盘通知；同一失效状态仅提醒一次，恢复后再次失效会重新提醒，点击通知可直接开始重新获取 Cookie。
- 设置窗口拆分为「账户与凭据」「运行行为」「软件更新」三个页签，保存操作固定在底部。
- DeepSeek 新增一键获取 Cookie；MiMo 与 DeepSeek 均使用独立浏览器配置目录采集并回填 Cookie。

### 修复

- 恢复点击其它应用导致面板失焦时自动收起并显示悬浮球的行为；打开设置、拖动面板或窗口重新激活时不会误收起。
- 修复 MiMo 悬浮球在收起状态下更新滞后的问题：可见时改用当前月的轻量请求，不再等待历史账单补同步。
- 统一面板与悬浮球的刷新频率，始终遵循设置的刷新间隔。

### 优化

- 粘贴或浏览器采集小米 MiMo Cookie 时，自动提取并回填 `api-platform_ph`；同时规范化多余空白，避免重复复制凭据或请求参数重复拼接。

### 验证

- 补充配置默认值与保存读取、设置项回显保存、面板失焦收起及保护条件的回归测试。
- 补充鉴权失效通知去重、通知跳转、DeepSeek Cookie 回填与浏览器 Cookie 域名过滤的回归测试。
- 补充 MiMo 轻量刷新仅请求当前月、跳过历史补同步，以及收起状态刷新间隔的回归测试。
- 补充面板与不同平台悬浮球均遵循配置刷新间隔的回归测试。
- 补充 `api-platform_ph` 提取、Cookie 回退为独立凭据字段和 Cookie 空白规范化的回归测试。
